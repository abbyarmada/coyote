Coyote Batch File Transfer

What it can do for you

Coyote BatchFT is a collection of file transfer utilities and tasks which can be added to the Coyote Batch tool kit to aid in creation of data extraction, transformation and loading jobs. While these file transfer tools are targeted to the Coyote Batch tool kit, these classes can be used in other components to speed the development process.

ToolKit, Not a Framework

This is a collection of tools which can be used individually or in concert to achieve the level of functionality desired. Just need to copy a directory form one location to another? Just use the components in this toolkit. For more structured execution, use the batch job to in Coyote Batch to execute the file transfer tasks.
