Coyote Batch MQTT Tools

What it can do for you

Coyote BatchMQTT is a collection of utilities and components which can be added to the Coyote Batch tool kit aiding in the exchange of data using the MQTT protocol to aid in creation of data extraction, transformation and loading (ETL) jobs. While these tools are targeted to the Coyote Batch tool kit, these classes can be used in other components to speed the development process.

ToolKit, Not a Framework

This is a collection of tools which can be used individually or in concert to achieve the level of functionality desired. Just need to connect to an MQTT server? Just use the components in this toolkit. Use the simple utilities in this library to keep your code simple an readable. For more structured execution, use the batch job to in Coyote Batch to exchange data in continuous manner.
