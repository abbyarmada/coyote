TinyDX

What it can do for you

TinyDX is a version of the CoyoteDX framework and toolkit to make running your components easier on any systems which supports the Java[tm] platform. It uses a very modular architecture that lets you add your own components so the tool works as you want, not as the software requires you to work. It is possible (and expected) to be extended it to suit your needs and has been designed with this purpose in mind from the start.
